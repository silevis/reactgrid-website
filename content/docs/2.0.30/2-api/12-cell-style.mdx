`CellStyle` interface prevents passing unwanted CSS properties that could broke grid rendering. Used in cell template.

## Definition

```tsx
export interface CellStyle {
    readonly color?: string;
    readonly background?: string;
    readonly overflow?: string;
    readonly paddingLeft?: string;
}
```

## Properties

| Property name     | Type      |  Property description         |
| -----------       | -------   | -------------------------     |
| `color?`          | `string`  | CSS `color` property          |
| `background?`     | `string`  | CSS `background` property     |
| `overflow?`       | `string`  | CSS `overflow` property       |
| `paddingLeft?`    | `string`  | CSS `padding-left` property   |
