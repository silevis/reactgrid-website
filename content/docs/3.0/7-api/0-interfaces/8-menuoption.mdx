Context menu is part of grid, each menu option must implement interface below. 

## Definition

```ts
interface MenuOption {
    id: string;
    label: string;
    handler: () => void;
}
```

## Properties

| Property name | Type          | Property description |
| -----------   | ------        | -------------------- |
| `id`          | `string`      | Text that identifies each menu option |
| `label`       | `string`      | Label displayed in menu |
| `handler`     | `() => void`  | Function that is called when option is clicked |
