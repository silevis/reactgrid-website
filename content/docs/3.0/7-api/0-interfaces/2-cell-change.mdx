`CellChange` interface is used by `onCellsChanged` callback in [`ReactGrid`](link) component. 

## Definition

```tsx
interface CellChange<TCell extends Cell = Cell> {
    readonly rowId: Id;
    readonly columnId: Id;
    readonly initialCell: TCell;
    readonly newCell: TCell;
}
```

`TCell` is type that extends common [`Cell`](link) type. It can be one of [predefined cell types](link) or one of yours. 
Read more about creacting [custom cell templates](link).

## Properties

| Property name     | Type                                      | Property description | 
| ------------      | -------------                             | -------------------- |
| `rowId`           | [`Id`](link)                              | Row [`Id`](link) where change ocurred | 
| `columnId`        | [`Id`](link)                              | Column [`Id`](link) where change ocurred | 
| `initialCell`     | `TCell extends Cell = `[`Cell`](link)     | Old content of cell | 
| `newCell`         | `TCell extends Cell = `[`Cell`](link)     | New content of cell |
